Masmorreiros é um sistema de RPG de mesa tradicional, jogado com papel, lápis e dados comuns (d6) que possui uma abordagem simplificada e um espírito Old School.

Ele foi criado para ser simples, rápido e modular. Aqui você vai encontrar tudo que precisa para jogar RPG rápido e fácil!
